This is the static version, to run the interactive version please follow the steps below:

  1. Install bokeh: pip install -U bokeh --user
    • Be aware bokeh needs to be of version 13.0 or higher for all elements on the bokeh run server to work.
  2. Open your terminal
  3. Enter: 'git clone git@github.com:DanielPerezJensen/bscKI-data-analyse.git'
  4. cd bscKI-data-analyse
  5. Enter: bokeh serve bokeh-server
  6. In your webbrowser navigate to: http://localhost:<**PORT**>/bokeh-server
  7. The port might be different on your machine, look in your shell to see the exact link.
